# Frontend Developer Technical Test

This project is a **Frontend Application** built as part of a **Frontend Developer Technical Test**.  
The application consumes data from a provided backend API and implements authentication, dashboard visualization, master data management, and reporting features.

## Steps to run FE
1. Clone this repository (git clone)

2. Change the working directory to the project (cd FE_Test)

3. Install the node module (npm install) in directory FE_Test

4. Start the FE server (npm run dev)

5. Open the default live: http://localhost:5173/
